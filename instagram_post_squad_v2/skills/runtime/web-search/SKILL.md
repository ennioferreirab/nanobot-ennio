# Web Search

Use this skill to ground decisions in current external evidence.

## Core rules

- Search before making claims about current market behavior.
- Use multiple queries when one query is too broad.
- Prefer authoritative or directly relevant sources.
- Capture the date window when freshness matters.
- Return usable evidence, not just raw links.

## Good output shape

- what was searched
- what was found
- why it matters
- evidence links
- confidence or limitations

## Anti-patterns

- generic research with no question in mind
- single-source conclusions
- outdated pages presented as current signals
