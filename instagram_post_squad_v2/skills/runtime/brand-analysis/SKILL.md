# Brand Analysis

This skill turns sampled brand assets into an operational brand brief.

## Input expectation

- directory of scraped Instagram assets
- captions and metadata
- optional website or public brand references

## Analysis dimensions

- visual system
- tone of voice
- caption structure
- recurring content pillars
- CTA behavior
- offer and conversion cues
- anti-patterns
- confidence and unknowns

## Mandatory classification

Every important statement should be one of:

- Observed
- Inferred
- Unknown

## Output shape

- Brand Snapshot
- Brand Brief
- Anti-patterns
- Confidence / Unknowns

## Anti-patterns

- inventing exact fonts or colors without evidence
- calling the brand premium, youthful, serious, etc. without concrete signals
- ignoring conflicts between captions and visuals
