# Instagram Copywriting

Use this skill when the strategy is already approved and the job is execution.

## Required inputs

- approved post specs
- approved brand brief
- approved copy research or pattern library

## Output block per postId

- `postId`
- `objective`
- `caption`
- `hook`
- `CTA`
- `hashtags`
- `rationale`
- optional `riskNote`

## Rules

- preserve the post objective
- keep the first line strong
- match the CTA to the objective
- protect brand tone
- vary hook structures across the batch

## Anti-patterns

- generic captions that fit any brand
- CTA mismatch
- hashtag stuffing
- changing the post angle because it feels more creative
