# Brand Consistency Checklist

Before generating an image, confirm:

1. Which brand assets are the highest-priority references?
2. Which colors must appear, and in what proportions?
3. Which visual traits are mandatory to preserve?
4. Does the composition fit the target aspect ratio?
5. Is the layout legible on mobile?
6. Does the prompt avoid introducing unrelated styles?
7. Are logo language and typography behavior protected when relevant?
